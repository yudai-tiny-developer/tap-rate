# YouTube Playback Rate Control

A lightweight Chrome extension that adds playback rate buttons directly to the YouTube player.

It is designed for fast, touch-friendly rate switching without opening YouTube's settings menu. The extension also supports keyboard shortcuts and lets you enable or disable individual rate buttons from the popup.

## Features

- Adds playback rate buttons to the YouTube player UI
- Works on `https://www.youtube.com/*`
- Includes preset speeds from `0.25x` up to `16x`
- Lets you show or hide each preset from the popup/settings screen
- Supports Chrome keyboard shortcuts through the extension commands API

## Usage

1. Open any YouTube video page
2. Look for the added playback rate buttons in the player controls
3. Click a button to switch the current video's playback speed
4. Open the extension popup to enable or disable individual rate buttons

The popup is also used as the extension's options page.

## Keyboard Shortcuts

To assign keys:

1. Open `chrome://extensions/shortcuts`
2. Find **YouTube Playback Rate Control**
3. Set the shortcuts you want for each command
